Allocate()

Obtains an instance from the pool.

Declaration
T Allocate()
Returns
Type Description
T

An instance from this pool.
Remarks

Instances returned from this method will have been either newly created with Make() or will be a previously idle object and have been activated with Activate(T).

By contract, clients must return the allocated instance using Free(T), or a related method as defined in an implementation or sub-interface.

The behaviour of this method when the pool has been exhausted is not strictly specified (although it may be specified by implementations). It is encouraged to throw a System.InvalidOperationException.
