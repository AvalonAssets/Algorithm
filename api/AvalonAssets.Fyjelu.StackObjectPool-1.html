Type Parameters
Name Description
T The type of objects held in this pool.
Remarks

Given a IObjectFactory<T>, this class will maintain a simple pool of instances. A finite number of idle instances is enforced, but when the pool is empty, new instances are created to support the new load. Hence this class places no limit on the number of active instances created by the pool.

Remarks

It limits the maximum number of idle instances to maxPoolSize, and initially allocating a container capable of containing at least initialPoolSize instances. The pool is not pre-populated. The initialPoolSize parameter just determines the initial size of the underlying container, which can increase beyond this value if maxPoolSize > initialPoolSize.

Remarks

If there are idle instances available on the stack, the top element of the stack is popped to activate, and return to the client. If there are no idle instances available, Make() is invoked to create a new instance.

All instances are Activate(T) before being returned to the client.
